Test Flow

Open this page on both fake domains and click between them. The tracker JS will decorate cross-domain links with ?cd_cid=..., and the server should keep them in the same session.

Step 1: Open on Maxim domain
http://test.getmaxim.ai:8000/test/cross-domain
Step 2: Click this link to cross to Bifrost (should append cd_cid)
http://test.getbifrost.ai:8000/test/cross-domain
Step 3: Click this link to cross back to Maxim
http://test.getmaxim.ai:8000/test/cross-domain
Step 4: Simulate external entry (new session)

Opens in a new tab with no referrer (simulates arriving from Google). This should create a NEW session. Then continue steps 2→3 from that tab.

Verify

After clicking through, run this SQL on your Neon staging DB:

-- Sessions: steps 1-3 should share one session_id
-- Step 4 should be a different session_id
SELECT id, client_id, first_visit, last_visit,
       entry_referrer_domain, is_external_entry
FROM visit_sessions
ORDER BY last_visit DESC LIMIT 5;

-- Visits: check session_id grouping
SELECT id, session_id, page_domain, referrer, source, timestamp
FROM visits
ORDER BY timestamp DESC LIMIT 10;

-- Events: same session_id across domains
SELECT id, session_id, event_type, page_domain,
       referrer_domain, source
FROM visit_events
ORDER BY timestamp DESC LIMIT 15;
