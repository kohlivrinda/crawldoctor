CrawlDoctor provides multiple tracking methods to capture AI crawler visits that traditional analytics miss:

🖼️ Pixel Tracking

1x1 transparent pixel that works with all crawlers, even those without JavaScript support.

📡 Beacon API

Advanced tracking with structured data and custom events for detailed analytics.

📁 Resource Tracking

Alternative tracking through CSS, JS, or JSON resource requests.

🔍 AI Detection

Sophisticated crawler detection using user agent patterns and behavioral analysis.

🚀 Quick Start - JavaScript Tracking

Method 1: Script Tag

Recommended JS-only snippet for tracking page views, clicks, and form submits:

<!-- CrawlDoctor JS tracker --> <script src="https://your-domain.com/track/js?tid=your-tracking-id" async></script>
💡 Pro Tip: Place this in your <head> so it loads early.

Method 2: Single-Page Apps

Include the script once in your root layout. It automatically tracks SPA navigation.

<!-- SPA-friendly: include once in root layout --> <script src="https://your-domain.com/track/js?tid=your-tracking-id" async></script>

⚠️ Implementation Notes

🔒 Privacy: JavaScript tracking collects interaction metadata; avoid capturing sensitive fields and update your privacy policy accordingly.
📈 Performance: The JS tracker loads async and won't block page rendering.
🌍 GDPR: While crawler tracking is generally acceptable, ensure compliance with local privacy regulations.

📋 Testing Your Implementation

Verify your tracking is working correctly:

  1. Check Network Tab: Look for requests to /track/js and /track/event in browser dev tools
  2. Monitor Dashboard: Log into your CrawlDoctor dashboard to see real-time data
  3. Test Different User Agents: Use browser dev tools to simulate different crawlers
  4. Verify Script Load: Ensure the JS tracker loads without errors
# Test loading the JS tracker curl "https://your-domain.com/track/js?tid=test-site" # Send a sample event curl -X POST "https://your-domain.com/track/event?tid=test-site" \ -H "Content-Type: text/plain" \ -d '{"event_type":"form_submit","page_url":"https://example.com/demo","data":{"form_values":{"email":"test@example.com"}}}' # Check if request was tracked curl -H "Authorization: Bearer YOUR_TOKEN" "https://your-domain.com/api/v1/analytics/realtime"

🛠️ API Integration

For programmatic access to your tracking data:

// Get analytics summary fetch('https://your-domain.com/api/v1/analytics/summary', { headers: { 'Authorization': 'Bearer YOUR_TOKEN' } }) .then(response => response.json()) .then(data => console.log(data)); // Export data fetch('https://your-domain.com/api/v1/analytics/export?format=csv', { headers: { 'Authorization': 'Bearer YOUR_TOKEN' } }) .then(response => response.blob()) .then(blob => { const url = URL.createObjectURL(blob); const a = document.createElement('a'); a.href = url; a.download = 'crawldoctor-data.csv'; a.click(); });
